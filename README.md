# CloudinaryUITests

1. Software requirements:
- Visual Studio (Free Community Edition - https://visualstudio.microsoft.com/pl/thank-you-downloading-visual-studio/?sku=Community&rel=16#
- GIT (https://git-scm.com/download/win)

2. Installation

2.1 Go to directory when you would like to clone repository and type "git clone https://github.com/baca90/CloudinaryUITests.git"
2.2 Open Visual Studio -> File -> Open -> Project/Solution and find CloudinaryUITests.sln
2.3 Click Build (it can take some time for the first time due to nuget packages downloading)

3. Running

3.1 Find Test Explorer window on the right side, click right mouse button on Test and click Run
